# Homebrew Tap for Sidecar

This is a [Homebrew](https://brew.sh/) tap for [Sidecar](https://github.com/Nikschavan/Sidecar), a permission management sidecar for Claude Code CLI.

## Installation

```bash
brew tap nikschavan/sidecar
brew install sidecar
```

## Updating

```bash
brew update
brew upgrade sidecar
```

## Uninstallation

```bash
brew uninstall sidecar
brew untap nikschavan/sidecar
```
